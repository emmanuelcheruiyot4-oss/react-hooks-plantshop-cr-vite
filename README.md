🌱 Plantshop

A React application that allows users to view, add, search, and manage plants. The app is connected to a backend API that stores plant data using JSON Server.

📌 Features
View all plants on page load
Add a new plant using a form (POST to backend)
Mark plants as “Sold Out” (PATCH request)
Search plants by name in real time
Responsive UI with component-based architecture
